# Claude Code Plugins — Jack Barrie

Plugin marketplace for Claude Code.

## Install

Add the marketplace:
```bash
claude plugin marketplace add jackbarrie1341/claude-plugins
```

## Available Plugins

### autoreason

Iterative quality optimization through adversarial multi-agent debate. Pits isolated agents against each other until nothing can beat the current version.

```bash
claude plugin install autoreason@jackbarrie
```

[Source](https://github.com/jackbarrie1341/autoreason)
